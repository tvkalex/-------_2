### Создание коммитов
Для создание коммитов используется команда *git commit*. Для этого в терминале с папкой-репозиторием необходимо написать команду *git commit -m "«сообщение к коммиту› "*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО***.
